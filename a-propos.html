A Propos


Madame KYMG est un projet de groupe réalisé dans le cadre de la formation Concepteur Développeur d'Applications, à Ada Tech School. L'équipe se compose de 4 apprenants: Ghislaine, Kai, Yoann et Margot.

Nous avons utilisé HTML et CSS pour la mise en page et JavaScript pour la partie dynamique du site.



Nous avons utilisé plusieurs API pour construire nos prédictions :
  • Nationalize : une API pour prédire la nationalité d'une personne à partir de son prénom (donne l'information sous le format code Pays par exemple 'FR'),
  • HappyAPI : une API pour convertir le format code pays en nom de pays,
  • Flagsapi : une API pour obtenir le drapeau du pays à partir de son code pays,
  • Agify : une API pour prédire l'âge d'une personne à partir de son prénom et de sa nationalité (correspond à l'âge moyen dans le pays concerné)
